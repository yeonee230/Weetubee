# 📹 Youtube Clone JS

## 설명
JS로 만든 Youtube Clone 사이트 입니다. 

## 홈페이지
http://weetubee.herokuapp.com/

## 개발 목표
* 풀스택으로 개발해보기
* 주요기능 완성하고 배포하기

## 사용 기술
* Front-end : Pug, JS, SCSS
* Back-end : NodeJS, Express
* DB : MongoDB
## 주요 기능
* 회원가입/로그인을 할 수 있습니다.  
* 영상을 업로드/수정/삭제 할 수 있습니다. 
* 댓글을 추가/삭제할 수 있습니다.
